# NotClicky

NotClicky is the creator of [[NotClicky]], a macOS menu-bar companion app for voice help, screen-aware guidance, cursor pointing, and background Agent Mode tasks.

NotClicky is built around local configuration, native macOS interactions, and a practical workflow where voice, text, and coding agents can work together from a lightweight desktop interface.
